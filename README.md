### 🤙 RULE

### 코딩테스트 진행
- 각자 코딩테스트 진행 및 풀이 완료시 개인 브런치로 PR
- PR 이후 각자 피드백 작성
  - 장미 -> 인준 피드백
  - 인준 -> 하림 피드백
  - 하림 -> 지원 피드백
  - 지원 -> 장미 피드백
- 풀이 완료시 표에 🔵
